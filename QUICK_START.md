# QUICK START GUIDE - 2 MINUTES

## WINDOWS USERS

### Step 1: Open PowerShell
- Press `Win + X`
- Click "Windows PowerShell" or "Terminal"

### Step 2: Navigate to folder
```powershell
cd C:\Users\YourName\Downloads\ChessGamePro
```

### Step 3: Compile
```powershell
.\compile.bat
```
Wait for "Compilation successful!"

### Step 4: Run
```powershell
.\run.bat
```

---

## LINUX/MAC USERS

### Step 1: Open Terminal

### Step 2: Navigate to folder

```bash
cd ~/Downloads/ChessGamePro
```

### Step 3: Compile

```bash
./compile.sh
```

### Step 4: Run
```bash
./run.sh
```

## FIRST TIME PLAYING

### 1. Create Account
- Username: minimum 3 characters
- Password: minimum 4 characters
- Click "CREATE NEW ACCOUNT"

### 2. Login
- Enter your username and password
- Click "LOGIN"

### 3. Select Game Mode
- Click "VS AI" card
- Click "PLAY" button
- Choose difficulty: Easy, Medium, or Hard

### 4. Play!
- Click your piece (White)
- Click where to move (yellow squares)
- Earn trophies by winning!

## COMMON ISSUES

**"javac not found"**
- Install JDK from: https://www.oracle.com/java/technologies/downloads/
- Restart terminal/PowerShell

**Images not showing**
- Make sure you're in ChessGamePro folder when running
- Check that assets/pieces.png exists

**Account not saving**
- Program will create data/ folder automatically
- Check folder permissions
